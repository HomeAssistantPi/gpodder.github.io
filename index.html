gPodder // Media aggregator and podcast client

gPodder is a simple, open source podcast client written in Python using GTK+. In development since 2005 with a proven, mature codebase.

The latest version is 3.9.1, released 2016-08-31. Read the release notes.

Get it

  • Windows: gPodder 3.9.1 (also as ZIP)
    You also need to install 32-bit Python 2.7 and PyGTK 2.24
  • macOS: gPodder 3.9.1
    Already includes Python and PyGTK
  • Linux/BSD: Install via the package manager
    Available in most distributions (Debian, Ubuntu, Fedora, ...)
  • Source code: github.com/gpodder/gpodder
